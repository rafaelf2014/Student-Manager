# Student Management App

A **desktop application built in C# with WPF** to manage student records, grading, and group assignments. Designed for instructors to easily organize, evaluate, and track students throughout the academic year.

---

## Features

- Browse and search student records efficiently
- Filter students by class, grade, or group
- Create and manage student groups for group assignments
- Evaluate students and track grades over time
- Intuitive GUI designed with WPF for ease of use

---

## Technologies Used

- **Language:** C#  
- **Framework:** WPF (Windows Presentation Foundation)  
- **Platform:** Windows Desktop  
- **Data Storage:** In-memory (future versions can integrate database support)
